# MopanProxy 进度文档

## 项目概述

将和彩云（中国移动云盘 yun.139.com）通过 Go 语言代理，提供 Web UI 管理界面和标准 WebDAV 服务器接口。

**核心功能**：
- 文件名 + 文件夹名 + 文件内容 **全部 AES-256-GCM 加密**
- 云端任何人看到的都是不可读的 hex 字符串
- 浏览器手动登录获取 Token，无需自动登录
- Web UI 文件管理（浏览/上传/下载/删除/新建文件夹）
- WebDAV 服务器（支持 Windows/macOS/Linux 映射为网络驱动器）
- Web UI 登录保护（用户名/密码 + Session Cookie）
- 全局认证 + CSRF 保护 + 安全响应头

## 服务器信息
- 项目路径: /opt/mopan-proxy/
- 服务: mopan-proxy.service (systemd)
- Web UI 端口: 9090
- WebDAV 端口: 9091（MopanFS 云盘模式）
- HTTPS: webdav.96110.li（nginx + Cloudflare）
- qBittorrent: qb.96110.li（端口 9092）

### 加密模块 API

```go
// 名称加密
enc.EncNameForFolder("Documents")  // → "E_b28ce2c8..."
enc.EncNameForFile("report.pdf")   // → "E_50db1161...bin"
enc.DecNameForFolder("E_b28ce2c8...") // → ("Documents", true)
enc.DecNameForFile("E_50db1161...bin") // → ("report.pdf", true)

// 内容加密
enc.Encrypt(plaintext)  // → [12B nonce][ciphertext][16B tag]
enc.Decrypt(ciphertext) // → plaintext
```

## 和彩云 API

### 文件操作

| 功能 | 路径 |
|------|------|
| 文件列表 | `POST /hcy/file/list` |
| 创建文件/目录 | `POST /hcy/file/create` |
| 获取下载链接 | `POST /hcy/file/getDownloadUrl` |
| 删除文件 | `POST /hcy/file/batchDelete` |
| 移动文件 | `POST /hcy/file/batchMove` |
| 重命名 | `POST /hcy/file/update` |
| 完成上传 | `POST /hcy/file/complete` |

### 上传三步流程

```
Step 1: POST /hcy/file/create → 获取 uploadUrl + fileId
Step 2: PUT uploadUrl          → 上传密文
Step 3: POST /hcy/file/complete → 确认完成
```

### Token

```
Authorization = base64("pc:" + phone + ":" + token)
有效期: 约 30 天
获取: 浏览器 F12 → Network → 复制 Authorization 头
```
